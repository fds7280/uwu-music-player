# Terminal Music Player

A feature-rich terminal-based music player with YouTube streaming support.

## Features
- Local MP3 playback
- YouTube search and streaming
- ASCII art album covers
- Progressive caching of streamed content
- Metadata display
- Keyboard controls

## Dependencies
- ncurses
- PipeWire
- libsndfile
- TagLib
- yt-dlp
- ffmpeg

## Building
```bash
mkdir build
cd build
cmake ..
make
