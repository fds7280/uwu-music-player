# UwU Terminal Music Player

A feature-rich terminal-based music player with streaming support, Using yt-dlp [work in progress]

## Features
- Local MP3 playback
- YouTube search and streaming
- ASCII art album covers
- Keyboard controls

## Dependencies
- ncurses
- PipeWire
- libsndfile
- TagLib
- yt-dlp
- ImageMagick 
- ffmpeg

## Building
```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
